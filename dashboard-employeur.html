Soumettez cette déclaration sur socialsecurity.be avant le début du shift du travailleur.

Comment soumettre :
1
Connectez-vous sur socialsecurity.be avec votre accès employeur (carte d'identité électronique ou itsme)
2
Choisissez Dimona → Nouvelle déclaration → Type FLX
3
Encodez les données ci-dessus. Si la réponse est OK → le travailleur peut commencer
4
Si réponse NOK → le travailleur n'est pas éligible aujourd'hui, contactez FlexMoney
⚠️ Le numéro NISS du travailleur n'est pas encore renseigné. Demandez-le lui directement avant la soumission. Format : 00.00.00-000.00
